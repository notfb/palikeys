General Information
This is a private, non-profit site.

It's primary indented user group is the Buddhist Monastic Community (Sangha).
You are welcome to use it, but we kindly request you do not harm yourself or others doing so.
The site is run on Google Firebase.
The source code is freely available under the 3-clause BSD License on Github.
We do not use Google Analytics or other client side tracking software on this site.
Google firebase collects personal data in logfiles etc, see Privacy and Security in Firebase

The only data this site stores are your high scores in this JSON format:
{{ '{"id": "A random UUID", username: "A username or nickname", "score": "A number of your high score"}' }}

ERROR: {{error}}
Your high score data is: {{ score | json }}
You might have to change your username by clicking the user icon ( account_circle ) on the top row to be able to delete it.
Anonymous high score data can not be deleted (this is the username everyone uses when they do not provide a custom username).
