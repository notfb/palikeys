How does it work?

Touch Typing

This program is supposed to help you learn typing faster with the ten finger typing system also known as touch typing.
Please place your fingers on the home row (the middle row on your keyboard) and start typing the characters in the text field. Most keyboards have a little physical marker where your index fingers should be placed (on an English or German keyboard the F and J key).
As you are typing the correct keys they get marked in green.
If you make an error you have to restart the lesson. The idea being that your typing speed will suffer a lot if you make any errors and this modes will help you learn not to make any errors. We know this can be a little frustrating at first, but we kindly request your patience and hope it will benefit your long-term happiness. You do not have to type the spaces or newlines, but you can if you want to.

Keyboard Layouts

You can change the keyboard layout used to generate the Lessons from QWERTY to a custom Pali layout in the select box below the text in the lesson view ( keyboard ). The current layout is shown on the button of the screen. The keys needed for this lesson are highlighted.
If you want to try the Pali layout without installing a custom layout on your computer, then you can choose a mapping from the select box with the text 'No Mapping'. Change it to 'QWERTY -> Pali Meat' to map an English Keyboard.
An Auto Hot Key config file etc. of the Pali layout can be found on github.

High Scores

You get one score point for every character typed correctly and a lot of points for completing a lesson successfully.
If you type than more than 20 characters correctly you get some points as well.

You do not have to finish a lesson to go the the next lesson. You can use the navigation buttons below the text area to skip lessons. You can complete a lesson as many times as you want or not at all. It's all up to you and what you what to learn. Lesson five and six use real words you can type on the home row instead of randomly generated ones.
Lesson 23 to 25 use some text from the Dīgha Nikāya Buddhist Sutta Number 34 - Sangīti Sutta.

Navigation

If you click the user icon ( account_circle ) on the top row you can choose a user name. Keep in mind you have to set the user name every time you reload the page (for now...).
There is a high score reachable via the top row high score icon ( bar_chart ) as well.
The lesson icon in the top row ( fitness_center ) starts a new training session.
